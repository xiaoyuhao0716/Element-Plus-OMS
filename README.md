# Element-Plus-OMS

Element-Plus-OMS 基础框架

vue3 , Element-Plus , 动态路由 , 按钮权限 , pinia , Mock , 搭建的基础框架

##### 目录

<pre>
├── src                       // 开发目录
│   ├── apis                  // 请求接口
│   ├── assets                // 一些资源文件
│   ├── components            // 各种组件
│   ├── mock                  // 模拟服务器响应
│   ├── router                // 路由配置及map
│   ├── store                 // pinia 状态管理器
│   ├── utils                 // 工具 包含 axios 封装
│   ├── views                 // 各种页面 
│   ├── App.vue               // 根组件
│   ├── main.js               // 入口js
│   ├── config.js             // 配置文件
├── package.json              // 项目配置文件
</pre>
